# Loop Engineer concept book

Open `index.html` directly in a browser. It is designed to work from a local
`file://` URL without a web server or network dependency.

## Decision files

- `index.html` — visual offline concept book and local shortlist control
- `PROPOSALS.md` — concise five-concept decision document
- `DEVPOST_SUBMISSION_DRAFT.md` — current official requirements and reusable submission copy
- `ASSET_PLAN.md` — CC0 3D asset candidates and import guardrails
- `AZURE_FLUX_PROMPTS.md` — generated-image provenance and exact selected prompts
- `assets/` — five Azure FLUX concept images and deterministic SVG infographics

The generated art is concept material, not gameplay evidence. Replace it with
real captures after a concept is selected and implemented.
